### Description

This study calculates and displays the Directional Movement Index , ADX , and ADXR studies all in the same Chart Region. The Length and Mov Avg Length Inputs of this study correspond respectively to the DX Length and DX Mov Avg Length Inputs of those studies.


#### Inputs

- Length

- Mov Avg Length

- ADXR Interval


#### Spreadsheet

The spreadsheets below contains the formulas for these studies in Spreadsheet format. Save these Spreadsheets to the Data Files Folder .

Open them through File >> Open Spreadsheet .

Note : The percent errors in the Directoinal Movement Index spreadsheet are larger than those found in spreadsheets for other studies because the DMI study uses a different rounding scheme.
