# Volume Value Area for Bars

- Description

- Inputs


### Description

The Volume Value Area for Bars study calculates and displays the Volume Value Area High (VVAH) and Volume Value Area Low (VVAL) for each bar in the chart. The Volume Value Area High and Volume Value Area Low are represented, by default, as left offset squares.

For information on how the Value Area is calculated, refer to the Calculations section on the TPO Profile Chart study page.

It is essential that you properly set the Tick Size for the symbol you are applying this study to. This is set in Chart >> Chart Settings . If the Tick Size is not set correctly, it can cause long chart data load times and inaccuracies with this study.


#### Inputs

- Value Area Percentage : This setting specifies the percentage for the Volume Value Area. The default is 70%.
