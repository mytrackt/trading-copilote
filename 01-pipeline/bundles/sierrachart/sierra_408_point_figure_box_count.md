# Point and Figure Box Count

- Description

- Inputs


### Description

This study displays a graph below the bottom of the main price graph indicating the number of boxes for each Point and Figure chart bar.

This study is only meaningful when the Bar Period Type is set to Point and Figure Bar .


#### Inputs

This study has no Inputs.
