# ICE Futures and Spot Prices

## Overview <a href="#overview" id="overview"></a>

StockCharts.com provides end-of-day (EOD) data for select futures traded through the Intercontinental Exchange (ICE). These include spot prices for Brent Crude, Coffee, Cocoa, Cotton, and Sugar. Users will see the word “spot” in the name for these symbols. Spot prices allow users to create long-term charts for a specific commodity.

## Details <a href="#details" id="details"></a>

**Symbol Group:** ICE

**Publisher:** Intercontinental Exchange

**Update Frequency:** End-of-day (EOD)

**Online Source:** [ICE](https://www.ice.com/index)

**Links to Current Symbols:** [ICE and ICE Futures](https://stockcharts.com/find/?q=ice+spot\&searchMode=symbolCatalog)
