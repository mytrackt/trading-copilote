Cette classification couvre l'ensemble des configurations techniques.
**TRADEX-AI C1** : Les signaux générés par TRADEX-AI doivent être typés selon ces 4 catégories pour guider la lecture d'Abdelkrim en Mode Manuel — le type de trade conditionne la gestion de position.

### D5592 — Support/Resistance holding : catégorie distincte des breakouts
🔵 **ÉCOLE** (Source : come_and_study_support_and_resistance_trades_with_us_this_we.md) : Le maintien d'un support ou d'une résistance (support/resistance holding) est une catégorie de trade distincte du breakout. Le trade gagnant est le maintien ; le trade perdant est la cassure. Les deux sont les deux faces d'un même niveau.
**TRADEX-AI C1** : Lors de la détection d'un test de niveau S/R sur GC/HG/CL/ZW, le moteur doit distinguer : setup "maintien attendu" (signal dans le sens du S/R) vs "cassure attendue" (signal dans le sens contraire) — deux configurations opposées sur le même niveau.
*Catégorie : structure_marche*

### D5593 — Chevauchement entre types de trades (overlaps)
