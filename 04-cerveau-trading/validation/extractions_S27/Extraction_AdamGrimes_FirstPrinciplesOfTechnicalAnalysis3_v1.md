### D5871 — Les tendances sont statistiquement plus susceptibles de continuer que de se terminer
🟢 **FAIT VÉRIFIÉ** (Source : first_principles_of_technical_analysis_3.md) : Une fois une tendance en mouvement, elle est statistiquement plus susceptible de continuer que de se terminer. Ce principe justifie fondamentalement les stratégies dans le sens de la tendance (with-trend trading).
**TRADEX-AI C1** : La règle 3/4 actifs tradables + 2/3 confirmation alignés de TRADEX-AI filtre précisément les situations où la tendance est confirmée sur plusieurs actifs — maximisant l'application de ce principe statistique.

### D5872 — Chercher à picker les tops et bottoms est une approche perdante long terme

**TRADEX-AI C1** : TRADEX-AI ne génère pas de signaux d'inversion de tendance sans confirmation multi-actifs. Les signaux CONTRE la tendance dominante exigent un score /10 plus élevé et sont traités avec méfiance.

### D5873 — Le trading dans le sens de la tendance pardonne les erreurs d'entrée
🟢 **FAIT VÉRIFIÉ** (Source : first_principles_of_technical_analysis_3.md) : Le trading dans le sens de la tendance est plus indulgent car la puissance de continuation de la tendance permet d'être légèrement tôt ou tard sur les entrées et de rester profitable. Beaucoup d'erreurs sont pardonnées si on est aligné avec la tendance.
**TRADEX-AI C1** : En mode Auto, les entrées with-trend peuvent tolérer un léger slippage d'exécution. Les entrées contre-tendance (range fade) exigent une précision d'entrée plus stricte.

Examiner ses instincts : si l'envie de fader la tendance est constante, il faut changer d'approche.
**TRADEX-AI C1** : Le prompt Claude niveau 3 doit commencer par la détermination de la tendance dominante sur chaque actif TRADING (GC/HG/CL/ZW) avant toute analyse des cercles d'intelligence.

### D5875 — Pattern impulsion + retracement + continuation = structure de base de la tendance
🟢 **FAIT VÉRIFIÉ** (Source : first_principles_of_technical_analysis_3.md) : En tendance, le pattern de base est : mouvement impulsif dans la direction de la tendance → retracement contre cette impulsion → nouveau mouvement de tendance dans la direction originale. Ce pattern est la structure fondamentale de toute tendance.
**TRADEX-AI C1** : Ce pattern impulsion/retracement/continuation est directement aligné avec la méthode Belkhayate (BGC + COG). Le COG identifie les zones de retracement vers lesquelles price gravite avant de reprendre la tendance.
*Catégorie : configuration*

### D5876 — Entrer sur le retracement contre-tendance est l'un des meilleurs trades techniques
🟢 **FAIT VÉRIFIÉ** (Source : first_principles_of_technical_analysis_3.md) : L'un des meilleurs trades techniques consiste à identifier une impulsion, entrer sur le retracement contre-tendance (pullback), et tenir la position lorsque le marché reprend la direction de l'impulsion originale. Plusieurs méthodes existent : se positionner dans le retracement ou entrer sur la reprise du momentum with-trend.
**TRADEX-AI C1** : Cette logique de pullback est centrale dans la méthode Belkhayate. L'Énergie Belkhayate (C1, stub actuel) doit mesurer l'épuisement du retracement pour optimiser le timing d'entrée.
