```

- Conditions : confiance >= 75% ET score >= 18/21 ET DD_jour < 2%

  * Claude API indisponible (fallback max 65% → Auto interdit)

BASCULE MANUEL → AUTO : bouton dans dashboard, confirmation requise

```
